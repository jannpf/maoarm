# maoarm

**Was wollt ihr denn?**  
  
Controlling the movement of the robotic arm RoArm-M2-S. In a cat-like way (whatever that is, we'll figure out).

## Basic usage

In 2 separate terminal windows, run the following scripts:

```sh
python3 -m cv
python3 arm/control.py
```
